# xiaofanshu小番薯

#### 介绍

#### 软件架构

基础框架：Spring Boot

持久性框架：Mybatis-Plus

MVC框架：Spring MVC

缓存：Redis

搜索引擎：ElasticSearch

关系型数据库：Mysql

非关系型数据库：MongoDB

日志：Logback+Slf4j

依赖管理：Maven

消息中间件：RocketMQ

分布式调度中心：xxl-job

微服务架构：

- Spring Cloud+Spring Cloud Alibaba
- 服务中心+配置中心：nacos
- 网关：gateway
- 事务统一：RocketMQ的事务消息（执行周期长且实时性要求不高的场景），不必要引入Seata
